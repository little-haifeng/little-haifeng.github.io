Sublime Text python 配置

little-feng · 2021-8-16 · 次阅读

前言

  本人固态硬盘干掉大约一年了，也没有买。每次开机2分钟差不多，没换，因为系统都刷在了机械盘，换系统的时候巨麻烦。干脆就没搞，但是那个Pycharm真是累，每次打开也需要一分钟，刷新文件。但是jupyter每次写代码都没有自动补全，对于我这种代码半吊子来说，巨累。累啦累啦。然后就想到了一款轻量化的编辑器，VScode没用过，就不想用。选择了以前傻瓜式的Sublime Text3，主要是打开巨快。对于我这种电脑身残志坚户的一种福利。其实吧，Jupyter也有很多插件，也能补全，但是补全功能不好用，不过他的那个代码自动对齐和一些PPT那些功能也不错，等一些会介绍一下Jupyter美化等，那赶紧废话不多说！赶紧配置一手。


一.安装sublime

Sublime Text:一款具有代码高亮、语法提示、自动完成且反应快速的编辑器软件,不仅具有华丽的界面,还支持插件扩展机制，最重要的是非常的轻便。

如果你以前安装过，然后环境搞得乱七八糟。请卸载掉，然后再在删除卸载后剩余文件,本人路径

F:\Sublime Text 3
C:\Users\33309\AppData\Roaming\Sublime Text 3
C:\Users\33309\AppData\Local\Sublime Text 3

sublime Texe3下载地址：http://www.sublimetext.com/3 选择windows64便携版即可食用，中途记得打一个对勾 Add to explore context menu


二.安装packageControl

Sublime text 是通过package control来对插件进行安装和卸载。你可把 package control 当成一个软件商店。

现在版本不需要手动下载或者输入一些爬虫代码请求了，所以在此就不介绍了。要是你的还要手动，肯定是环境卸载不干净。


三. 搭建python运行环境

1、打开Tools > Build System > New Build System..

2、点击New Build System后，会生成一个空配置文件，在这个配置文件内覆盖配置信息，本文python安装路径为“D:/python37”，（注意区分正反斜杠，请将路径换成python实际安装路径），然后按ctrl+s，将文件保存在默认路径，文件名命名为“Anaconda.sublime-build”

{
    "cmd": ["F:/Anaconda/python.exe","-u","$file"],
    "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",
    "selector": "source.python",
    "shell": "true"  // 这个是后来matplotlib 画图不显示问题
}

之后就可以编译快乐的跑代码了，本人快捷键是Ctrl+B和F7。


四.插件Anaconda

安装方法

1、Preferences->Package Settings->Package Control->Install package,下载Anaconda，要等一会，低下等号在不断的移动。

2、Preferences->Package Settings->Anaconda->Settings User，添加如下内容

{
    "python_interpreter":"F:\\Anaconda\\python.exe",
    "suppress_word_completions":true,
    "suppress_explicit_completions":true,
    "comlete_parameters":true,
    "swallow_startup_errors":true,
    "anaconda_linting":false
}

之后就基本上配置好了，代码自动补全。还有各种各样的插件，比较的html。我这边只是最基础配置


五.更换主题

同样在Package Control：Install package 里面下载Material 快捷键Ctrl + Shift + p 。之后在选择。有4个样式。
