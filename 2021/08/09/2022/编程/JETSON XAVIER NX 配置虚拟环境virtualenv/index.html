JETSON XAVIER NX 配置虚拟环境virtualenv

little-feng · 2021-8-9 · 次阅读

Python虚拟环境

我们将使用python的虚拟环境，来保证你使用的环境互相独立。
使用virtualenv和virtualenvwrapper,(之前尝试过anaconda,由于各种包编译的问题 ，安装过程不是很顺利，为了快速使用，没有去做过深研究），安装命令：

pip3 install virtualenv virtualenvwrapper

安装完之后，需要更新.bashrc

gedit ~/.bashrc

在文件底部输入

# virtualenv and virtualenvwrapper
export WORKON_HOME=$HOME/.virtualenvs #指定所有的需拟环境的安装位置
export VIRTUALENVWRAPPER_PYTHON=/usr/bin/python3 #指定解释器
source /usr/local/bin/virtualenvwrapper.sh #进行激活生效

保存并退出，使用source 重新载入

source ~/.bashrc

接着创建python的虚拟环境，

mkvirtualenv yolo -p python3.6 #-p 是指定解释器是python3.6
python -m pip install --upgrade pip # 升级pip pip类似软件商店

创建后直接进入了。我的名字是yolo,你可以自己定义。

一些virtualenvwrapper的常用命令：举例如下

#创建一个环境
mkvirtualenv yolo -p python3.6 #在~/.virtualenvs下安装name的需拟环境
#激活工作环境  就是在该环境下工作
workon yolo
#workon后不加任何东西可以列出所有虚拟环境
#退出当前虚拟环境
deactivate
#删除虚拟环境，需先退出
rmvirtualenv yolo
#列出所有虚拟环境
lsvirtualenv

注意事项

网络连接失败

大概你会遇到这种问题，网速不行连接失败

虽然只有10KB/s，能别换就别换源，以免到以后报错一堆，在这个问题上大概可以换，可能以后安装其他包可能会报错，这个问题我重复执行了很多次，然后reboot重启一下机器，网速稍微好一点，装上去了。不过很多人都换源了，只是可能会日后错误。更改pip源即可。无需修改apt-get源。
